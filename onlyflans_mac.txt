Forma numero 1
Requiere instalacion de virtualenvwrapper
Creación de entorno virtual django en mac

1-	mkvirtualenv vOnlyflans
2-	workon vOnlyflans
3-	workon (verificar todos los entornos virtuales creados)
4-	Python3 –version (verificar versión instalada, captura de pantalla)
5-	Pip3 install django==3.2
6-	Python –m django versión  o Python –m django-admin versión  (verificar versión de django instalada en nuestro entorno virtual)
7-	Pip3 freeze > requirements–vOnlyflans.txt

Creación de proyecto django en mac

1-	django-admin startproyect OnlyFlans
2-	cd OnlyFlans
3-	python3 manage.py startapp onlyflans_app  (web)
4-	INSTALLED_APPS = [
“onlyflansapp”,
…
]
5-	revisar view.py
6-	revisar urls.py
7-	python3 manage.py makemigrations
8-	python3 manage.py migrate
9-	python3 manage.py runserver
